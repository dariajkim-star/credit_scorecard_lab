# 손익 기반 Cutoff — 경영진 1페이저 (Story 2.4)

> **이 값은 손익 시뮬레이션이며 실제 재무 데이터가 아니다.** 검증 표본(2015 빈티지, OOT)의 실측 상환·회수 데이터(원 데이터셋 Lending Club, **미국 달러(USD)** 기준)를 가정된 평균 대출금액($12,000)으로 스케일링한 결과다. 원화 환산 없이 데이터 원 통화 그대로 보고한다 — 국내 통화로 환산해 논의하려면 시점별 환율을 곱해 재계산할 것.

## 한 줄 요약

**현재 cutoff(546점)를 리스크 지표(부도율) 기준으로 정했지만, 손익 기준으로는 더 낮은 cutoff(챔피언 494점 / 챌린저 507점)이 연간 기대손익을 더 높인다.** 부도율은 낮아지지만, 그만큼 승인 가능했던 "순이익 창출 대출"도 함께 거절하고 있었다는 뜻이다.

## 수치 (avg_loan_amnt = $12,000 가정)

| | 챔피언(로지스틱) | 챌린저(LightGBM) |
|---|---|---|
| 현재 cutoff | 546.0 | 547.6 |
| 현재 승인율 | 47.7% | 41.4% |
| 현재 연간 기대손익 | $130M | $112M |
| **최적 cutoff** | **494.4** | **507.1** |
| 최적 승인율 | 100.0% | 100.0% |
| **최적 연간 기대손익** | **$261M** | **$261M** |
| **손익 delta** | **+$132M (+102%)** | **+$149M (+132%)** |
| 승인율 delta | +52.3pp | +58.6pp |

*(검증 표본 규모 기준 절대값 — "가정" 절 참고, 실제 포트폴리오 규모로 스케일링 필요. avg_loan_amnt=$12,000은 원 데이터셋 평균 대출금액(~$12,639)에 근사한 예시값.)*

## 왜 이런 결과가 나오는가

이 데이터셋(2012~2015년 승인 대출, 36개월물)에서는 **이자수익이 부도손실을 평균적으로 상쇄하고도 남는 구간이 승인 거절 영역까지 뻗어 있다** — 즉 리스크 관점에서 "위험하다"고 걸러낸 대출 중 상당수가 손익 관점에서는 여전히 순이익을 낸다(연 이자율이 높은 대출일수록 개별 부도 확률은 높아도 정상 상환 시 수익이 크기 때문). 두 모델(로지스틱·LightGBM) 모두 같은 방향의 결론을 낸다는 점이 이 발견의 신뢰도를 뒷받침한다.

## 가정 (assumptions — API 응답에도 항상 포함됨)

- 연간 볼륨은 OOT 검증 표본(2015 빈티지, 283,026건) 규모를 그대로 1년치 승인 볼륨으로 가정(별도 확대 계수 없음) — 실제 포트폴리오 규모에 맞춰 재스케일링 필요.
- 평균 대출금액은 요청 파라미터로 스케일링하며, 실제 승인 건별 대출금액 분포(원 데이터셋 최소 $1,000~최대 $35,000)는 반영하지 않는다.
- 회수율·상환액은 검증 표본의 실측치를 그대로 사용하며, 향후 금리·매크로 환경 변화는 반영하지 않는다.

## 한계 (정직하게 밝힘)

- 모형이 7개 변수만 사용하고 등급(grade)·금리(int_rate)를 의도적으로 배제(Epic 1 설계, 순환논리 방지 목적)한 축소 모형이라, "거의 전원 승인이 최적"이라는 결론이 **실제 상용 스코어카드(더 많은 변수·최신 데이터)에서도 그대로 성립할지는 별도 검증이 필요**하다.
- 이 결론은 컨설팅 시연용 방법론 증명이지, 즉시 실행 가능한 여신정책 권고가 아니다 — 방법론(risk-cutoff와 profit-cutoff를 나란히 비교하는 프레임)이 핵심 산출물이다.
- 원 데이터가 미국 달러 표시라 위 수치도 달러 그대로 보고했다 — 국내 청중 대상 발표 시 환율 환산 여부를 별도로 결정할 것(이 스토리 스코프 밖).
