# 비금융 텍스트 파생변수 검증 리포트 (컨설턴트 킥③, Story 3.2)

> **한 줄 결론**: 직함(emp_title) 텍스트에서 파생한 변수의 IV는 **0.0116으로 업계 관행 임계(0.02) 미만**이며, 7개 정형 변수 전부보다 낮았습니다. **이 데이터·이 방식에서 비금융 텍스트는 유의미한 추가 판별력을 주지 못했습니다.** — 그리고 이 "검증했다"는 사실 자체가 산출물입니다(효과 유무와 무관).

## 무엇을 검증했나

원본 데이터의 `emp_title`(직함, 자유 텍스트)이 신용 위험 신호를 담고 있는지를, **정교한 NLP 없이** 검증했습니다.

- **정제**: 소문자화 → 특수문자를 공백으로 → 다중공백 정리 → 앞뒤 공백 제거. (어간추출·임베딩·불용어·동의어 사전 등 정교한 NLP는 의도적으로 배제 — AC 명시.)
- **카테고리 매핑**: 학습(train) 표본 기준 **상위 20개 빈도 직함**을 각자 카테고리로, 나머지는 `OTHER`, 결측은 `MISSING`. 빈도는 train에서만 정하고 적용(누수 방지).
- **WOE/IV**: 기존 스코어카드 비닝 경로(`binning.py`, optbinning categorical solver)를 그대로 재사용(AD-2 단일 경로) — 새 공식 재구현 없음.

## 결과 (train 143,892건, 부도율 12.70%)

| 변수 | IV | 판별력 (관행: IV≥0.02) |
|---|---:|---|
| fico_range_low | 0.1298 | 강 |
| annual_inc | 0.0986 | 중 |
| dti | 0.0422 | 약 |
| home_ownership | 0.0346 | 약 |
| revol_util | 0.0297 | 약 |
| inq_last_6mths | 0.0284 | 약 |
| purpose | 0.0244 | 약 |
| **emp_title_category** | **0.0116** | **미달 (예측력 없음)** |

**emp_title 파생변수는 IV 0.0116으로 임계(0.02) 미만이며, 모든 정형 변수보다 낮습니다.**

## 왜 신호가 약한가 (해석)

1. **극단적 롱테일**: 직함 고유값이 207,516개(결측 6.7%)라, 상위 20개로 축약해도 **표본의 88.8%가 `OTHER`** 한 덩어리로 들어갑니다. 즉 대부분의 신청 건에 대해 이 변수는 "그 외"라는 정보 없는 값이 됩니다.
2. **직함 ≠ 표준화된 범주**: 상위 빈도에 직종(teacher, registered nurse)과 **고용주(us army, bank of america, walmart)가 섞여** 있습니다. Lending Club의 emp_title은 자유 입력이라 직종·고용주·약어(rn ↔ registered nurse)가 혼재해, 단순 빈도 매핑으로는 일관된 의미 축을 만들지 못합니다.
3. **소득·연체 등에 이미 반영**: 직업의 위험 관련 정보(안정성·소득 수준)는 상당 부분 `annual_inc`·`dti` 같은 정형 변수에 이미 흡수돼 있어, 직함 텍스트가 더할 한계 정보가 작습니다.

## 시사점 (컨설팅 관점)

**"비금융 텍스트를 무작정 넣지 않고 검증한 뒤 판단했다."** 결과적으로 이 데이터·이 방식에서는 직함 파생변수를 모형에 편입할 근거가 약합니다(IV 임계 미달). 이는 실패가 아니라 **검증 결과**입니다 — 근거 없는 피처를 넣어 모형을 복잡하게 만드는 것을 피하는 판단 자체가 가치입니다. 텍스트를 살리려면 (a) 표준 직종 코드 매핑(외부 사전) 또는 (b) 의미 임베딩 같은 정교한 접근이 필요하나, 이는 본 검증의 스코프(단순·투명) 밖이며 기대 효과 대비 비용을 별도 평가해야 합니다.

## 가정과 한계 (정직성 원칙)

- **정교한 NLP 미적용**(의도적): 직함의 의미론적 정보(직종 유사성)는 활용하지 않았습니다. 단순 빈도 매핑의 한계이며, 이것이 IV를 과소평가했을 가능성은 있으나 스토리 취지(억지 효과 배제)상 단순 방식을 고수했습니다.
- **결측 6.7%**는 `MISSING` 별도 카테고리로 보존(WOE는 empirical, 조용히 버리지 않음).
- **상위 20개 커버리지 낮음**(OTHER 88.8%): 더 세분화하면 각 빈이 통계적으로 얇아져 과적합 위험이 커지므로 확장하지 않았습니다.
- IV 비교는 동일 train 표본·동일 코드경로(`fit_binning`+`iv_table`)로 재산출한 값입니다(사과 대 사과).
- emp_title을 챔피언/챌린저 모형에 **실제 편입하지 않았습니다** — 이 스토리는 IV 기여도 검증만(편입은 별도 의사결정).
